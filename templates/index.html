Features

  • ✅ High-performance web framework with Fiber
  • ✅ In-memory index caching with TTL
  • ✅ File caching with LRU eviction
  • ✅ Content compression (gzip/deflate)
  • ✅ Both JSON and HTML API support
  • ✅ Compatible with pip, poetry, pipenv

Quick Start

Install packages via groxpi:

pip install --index-url http://localhost:5002/index/ your-package

Browse packages:

View Package Index

Cache Management:

  • Clear package list cache: curl -X DELETE http://localhost:5002/cache/list
  • Clear specific package cache: curl -X DELETE http://localhost:5002/cache/PACKAGE_NAME

API Endpoints

  • GET / - This home page
  • GET /index/ - List all packages (JSON/HTML)
  • GET /index/{package}/ - List package files (JSON/HTML)
  • GET /index/{package}/{file} - Download package file
  • DELETE /cache/list - Invalidate package list cache
  • DELETE /cache/{package} - Invalidate package cache
  • GET /health - Health check endpoint
